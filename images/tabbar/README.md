# TabBar 图标说明

此目录用于存放底部导航栏的图标文件。

## 需要的图标文件

请在此目录下添加以下图标文件（建议尺寸：81px × 81px）：

### 首页
- `home.png` - 首页未选中图标
- `home-active.png` - 首页选中图标

### 附近钓场
- `nearby.png` - 附近钓场未选中图标
- `nearby-active.png` - 附近钓场选中图标

### 会员码
- `member.png` - 会员码未选中图标
- `member-active.png` - 会员码选中图标

### 垂钓记录
- `records.png` - 垂钓记录未选中图标
- `records-active.png` - 垂钓记录选中图标

### 个人中心
- `profile.png` - 个人中心未选中图标
- `profile-active.png` - 个人中心选中图标

## 图标要求

- **格式**：PNG（支持透明背景）
- **尺寸**：建议 81px × 81px（小程序会自动缩放）
- **颜色**：
  - 未选中状态：使用灰色（#7A7E83）
  - 选中状态：使用主题色（#667eea）
- **样式**：建议使用线条图标或填充图标，保持风格统一

## 图标设计建议

1. **首页图标**：可以使用房子🏠、首页📱等图标
2. **附近钓场图标**：可以使用地图📍、位置🎯等图标
3. **会员码图标**：可以使用会员卡💳、二维码📱等图标
4. **垂钓记录图标**：可以使用记录本📝、列表📋等图标
5. **个人中心图标**：可以使用用户👤、设置⚙️等图标

## 图标获取方式

1. **图标库推荐**：
   - [IconFont](https://www.iconfont.cn/) - 阿里巴巴图标库
   - [Iconfinder](https://www.iconfinder.com/) - 图标搜索
   - [Flaticon](https://www.flaticon.com/) - 免费图标库

2. **设计工具**：
   - 使用 Figma、Sketch、Adobe XD 等设计工具制作
   - 导出为 PNG 格式，支持透明背景

## 注意事项

- 如果没有图标文件，tabBar 仍然可以正常工作，只是不会显示图标，只显示文字标签
- 图标路径是相对于项目根目录的
- 确保图标文件名与配置中的路径完全一致
- 建议使用 2x 或 3x 尺寸的图标以确保在高分辨率设备上清晰显示
